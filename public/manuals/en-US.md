# ACIA-2 AI Real-Time Intelligent Interpreter - User Manual

Welcome to ACIA-2, your premium real-time translation and meeting collaboration platform.

## 1. Getting Started - Gemini API Key 🔑
To enable the full power of AI translation, you need a Gemini API Key.
- Go to **Settings ⚙️** in the top right corner.
- Enter your **Gemini API Key**.
- If you don't have one, you can get it for free from [Google AI Studio](https://aistudio.google.com/app/apikey).
- If the room creator is sharing their key, you might not need to enter your own.

> [!NOTE]
> Using your own API key ensures you have dedicated quota and better performance.

## 2. Meeting Rooms 👥
- **Create a Room**: Click "Create New Room" to start a session.
- **Join a Room**: Enter the Room ID or scan the QR Code shared by others.
- **Sharing**: Click the **Share** button to copy the invite URL or show the QR Code.

## 3. Translation & Voice 🎙️
- **Start Interpreting**: Click the microphone icon to begin real-time speech-to-text and translation.
- **Bilingual Display**: The system shows both original and translated text simultaneously.
- **Audio Output**: Enable "Audio Output" to hear the translation spoken aloud by high-quality AI voices.

## 4. Advanced Features 🛠️
- **Admin Settings**: Customize the header title, adjust AI responsiveness, and select preferred voice types.
- **Clear Transcripts**: The room owner can clear all transcripts for a fresh start.
- **Dark Mode**: Toggle between light and dark modes for comfortable viewing.
