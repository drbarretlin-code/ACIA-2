# ACIA-2 AI リアルタイム・インテリジェント翻訳システム - 操作説明書

ACIA-2へようこそ。プレミアムなリアルタイム翻訳と会議コラボレーションプラットフォームです。

## 1. はじめに - Gemini API キー 🔑
AI翻訳の全機能を有効にするには、Gemini APIキーが必要です。
- 右上の **設定 (Settings ⚙️)** を開きます。
- **Gemini API キー** を入力します。
- キーをお持ちでない場合は、[Google AI Studio](https://aistudio.google.com/app/apikey) から無料で取得できます。
- ルーム作成者がキーを共有している場合は、入力しなくても使用できる場合があります。

> [!NOTE]
> ご自身のAPIキーを使用することで、専用のクォータとより高いパフォーマンスを確保できます。

## 2. 会議室機能 👥
- **ルーム作成**: 「新しいルームを作成」をクリックしてセッションを開始します。
- **ルーム参加**: ルームIDを入力するか、共有されたQRコードをスキャンします。
- **共有**: **共有 (Share)** ボタンをクリックして招待URLをコピーするか、QRコードを表示します。

## 3. 翻訳と音声 🎙️
- **音声認識開始**: マイクアイコンをクリックして、リアルタイムの音声文字起こしと翻訳を開始します。
- **2ヶ国語表示**: 原文と翻訳文が同時に表示されます。
- **音声出力**: 「音声出力」を有効にすると、高品質なAI音声で翻訳文が読み上げられます。

## 4. 高度な機能 🛠️
- **管理者設定**: ヘッダータイトルのカスタマイズ、AIの応答速度の調整、好みの音声タイプの選択が可能です。
- **履歴のクリア**: ルームオーナーは、すべての文字起こしをクリアして新しく開始できます。
- **ダークモード**: ライトモードとダークモードを切り替えて、快適に閲覧できます。
