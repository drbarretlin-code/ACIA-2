# ACIA-2 AIリアルタイム翻訳システム ユーザーマニュアル

---

## 1. インターフェース機能と操作ガイド

### 1. システムへのログイン

初めてウェブページを開いた後、以下の手順を完了してください：

1. **名前を入力**：表示名を入力します（例：`Barret`、役職名、または座席番号）。
2. **Gemini API Keyを入力**：Google AI Studio APIキーを入力します（形式：`AIza...`）。
3. **ルームの作成または参加**：
   - 「新しいルームを作成」をクリックして翻訳セッションを開始します。
   - または、他の人が共有した招待リンクをURLバーに貼り付けて直接参加します。

> [!NOTE]
> APIキーはブラウザのローカルに保存され、サーバーにアップロードされることはありません。

---

### 2. 言語選択バー（コントロールパネル）

画面中央の横列、左から右の順序：

| 要素 | 説明 |
|---|---|
| 🏳️ 左側の国旗 | あなたが話す言語（ソース言語）|
| 🌐 左側のドロップダウン | 「マイ言語」（Local）を選択、録音中は変更不可 |
| ⇄ 中央の矢印ボタン | 左右の言語を素早く切り替え、録音中は操作不可 |
| 🌐 右側のドロップダウン | 「相手の言語」（Client）を選択、録音中は変更不可 |
| 🏳️ 右側の国旗 | 相手の言語の国旗 |
| 🔴 **LIVE 赤色バッジ** | リアルタイム音声接続正常、Gemini Live APIで翻訳中 |
| ⚡ **Local アンバーバッジ** | フォールバックモード、ローカル音声認識使用中、翻訳は正常動作 |
| 🎙️ 録音ボタン | 🔵 青 = 未録音、クリックで開始；🔴 赤枠点滅 = 録音中、クリックで停止 |

> [!IMPORTANT]
> 言語は**録音を開始する前に**設定を完了する必要があります。録音中は切り替えできません。

---

### 3. 翻訳チャットボックス

- 各メッセージには、話者名（例：`Barret`）、原文、翻訳文が表示されます。
- 上部のステータスバー：
  - 🔴 **リスニング中...** = Gemini Liveがリアルタイムで音声処理中
  - 🟡 **ローカルモードリスニング中...** = フォールバックモード中、認識と翻訳は正常動作
- 右上のボタン：
  - **共有**：すべての会話記録をコピーまたは共有
  - **クリア**：すべての会話記録を削除（ルーム作成者のみ操作可能）

---

### 4. テキスト入力バー（画面下部）

音声が使用できない場合、または正確なテキスト入力が必要な場合：

1. 左側の方向ボタンをクリックして翻訳方向を切り替え（Local → Client または Client → Local）。
2. テキストボックスにメッセージを入力。
3. EnterキーまたはSendボタンをクリックして翻訳。

> [!NOTE]
> テキスト入力はいつでも使用可能です。**録音を先に開始する必要はありません**。

---

### 5. 音声出力設定（右下フローティングボタン）

右下の円形ボタン（スピーカーアイコン）をクリックして再生設定を開きます：

| オプション | 説明 |
|---|---|
| **ミュート** | 翻訳の音声再生なし |
| **自分のみ** | 自分のデバイスでのみ翻訳音声を再生 |
| **全員** | すべての参加者が音声再生を聞ける |
| **他の人のみ** | 他の参加者のみ聞こえ、自分には再生されない |

---

### 6. 上部ツールバー

左上にブランドロゴ **TUC**（赤太字）とサブタイトル（デフォルト：Equipment Department）が表示されます。右側のツールバー（順序）：

| アイコン | 機能 |
|---|---|
| 📋 ルームID + コピーボタン | 招待リンクをコピーして他の人と共有 |
| 📱 QRコードアイコン | QRコードを表示して他の人がスキャンして参加 |
| 🚪 ログアウトアイコン | 現在のルームを退出（ルーム作成者は「会議室を終了」を選択可能）|
| 👥 ユーザー数（xx/100）| 現在のオンラインユーザー数 |
| 🌙 / ☀️ 月 / 太陽アイコン | ダーク / ライトモード切り替え |
| 🔒 ロックアイコン | 管理者設定（ルーム作成者のみ表示）|
| 🟢 **システム準備完了** | システムは正常に動作中 |

---

### 7. 管理者設定（ロックアイコン）

ルーム作成者のみアクセス可能、含まれる内容：

- **Quotas Limitation**：現在のAPIキーのRPM / RPD使用量ダッシュボードを表示。
- **APIキー設定**：Gemini APIキーまたはGoogle Cloud APIキーの変更。
- **APIティア切替**：Free Tier（無料）またはTier 1（従量課金）のクォータ対応。
- **ヘッダータイトル設定**：左側ブランド名（デフォルト：`TUC`）とサブタイトル（デフォルト：`Equipment Department`）のカスタマイズ。

## 2. 使用制限

### 1. APIクォータ制限

本システムはGoogle Gemini APIを使用しています。Free Tier（無料）アカウントの制限は以下の通りです：

| 項目 | Free Tier制限 |
|---|---|
| **1分あたりの接続回数（RPM）** | 2回 |
| **1日あたりの接続回数（RPD）** | 50回 |
| **単一接続の最大時間** | 3時間（システムが強制切断）|
| **アイドル自動切断** | 1時間後に続行を確認、3分間応答なしで自動切断 |

クォータリセット時間：
- RPM：毎分ローリングリセット（固定時間ではない）
- RPD：毎日タイ時間15:00（冬季）または16:00（夏季）にリセット

|---|---|---|---|

| フォールバックローカル音声認識 | 対応 | 対応 | **非対応** |
| テキスト入力翻訳 | 対応 | 対応 | 対応 |
| QRコード共有 | 対応 | 対応 | 対応 |

> **Firefoxユーザーはテキスト入力モードのみ使用可能です。**音声機能（リアルタイムおよびフォールバック）は利用できません。

### 3. 言語サポート

システムは37言語をサポートしています。ただし、**フォールバックモード（ローカルSTT）**では一部の言語で認識精度が低下する場合があります。以下の言語ではChromeまたはSafariの使用をお勧めします：
- アラビア語
- ヘブライ語
- フィリピノ語

---

### 4. iOSデバイスに関する注意事項

> [!WARNING]
> **アプリをバックグラウンドに切り替えるまたは画面をロックした場合**：マイクがiOSシステムにより中断されます。アプリに戻った後2秒以内に自動的に録音が再開されます。自動的に再開されない場合は、録音ボタンを手動でクリックしてください。

- **通話中**：録音機能は一時停止します。通話終了後に手動で録音を再開してください。

## 3. 通常の使用シナリオとクォータ制限の回避

### 基本情報

Gemini Live音声の「接続回数」の計算方法：**録音を開始するたび、またはシステムが自動再接続するたびに1リクエストとしてカウント**されます。話した時間の長さとは無関係です。

- **1回の連続会議**（1回録音開始、途中切断なし）= **1 RPD**を消費
- **1回の自動再接続** = 追加で**1 RPDと1 RPM**を消費

---

### 典型的な使用シナリオ

| 行動 | 消費回数 |
|---|---|
| 1回録音開始、3時間連続使用 | 1 RPD |
| 3時間期限到達、システム切断後手動再開 | +1 RPD |
| 言語切替後に再開 | +1 RPD |
| インターネット瞬断、システム自動再接続成功 | +1 RPD（自動）|

**1日あたり50 RPDの見積もり：**

各会議で平均2-3回の接続（再接続含む）を消費した場合、1日あたり約**16-25回の会議**が可能です。

---

### 制限に到達しないための推奨事項

> [!IMPORTANT]
> **録音開始前に言語を設定**：言語切替は再接続をトリガーし、切替ごとに+1 RPD。録音中は言語セレクターがロックされるため、先に録音を停止する必要があります。

> [!IMPORTANT]
> **1分以内に録音の開始/停止を2回以上繰り返さない**：Free Tierは1分あたり2回の接続のみ許可。3回以上開始/停止を繰り返すと、APIが「API一時的に利用不可」と拒否し、1分待つ必要があります。

1. **安定したインターネット接続を維持**：断続的な接続は自動再接続をトリガーします。Wi-Fiはモバイルデータより安定しています。

2. **長時間の会議では連続録音を推奨**：一時的に話を止めただけでは録音を停止しないでください。新しいセッション開始時のクォータ消費を避けます。

3. **⚡ Local（アンバーバッジ）の表示は正常です**：システムがローカル認識に切り替えたことを意味します。**翻訳は引き続き機能します。**バックグラウンドでクラウド再接続を試み、成功すると自動的に 🔴 LIVE モードに戻ります。

### ステータスメッセージの説明

| ステータスメッセージ | 意味 | 推奨アクション |
|---|---|---|
| 「翻訳セッションが終了しました。自動再接続中。」| セッションが正常に期限切れ（長時間使用後に一般的）| 自動再接続を待つ |
| 「API一時的に利用不可（制限に達した可能性）、ローカルモードに切替。」| RPDまたはRPM制限に到達 | 次の分または翌日のクォータリセットを待つ |
| 「接続が一時的に中断、ローカルモードに切替。音声とテキスト入力は引き続き使用可能。」| 不安定なインターネットによる切断 | 使用を続行;システムが自動的に回復を試みます |
| 「マイクが中断されました。録音を再開してください。」| iOSシステムがマイクを中断 | 録音ボタンをクリックして再開 |
