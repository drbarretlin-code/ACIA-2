# ACIA-2 AI 实时智能翻译系统 - 操作说明书

欢迎使用 ACIA-2，您的顶级实时翻译与会议协作平台。

## 1. 开始使用 - Gemini API Key 🔑
为了启动完整的 AI 翻译功能，您需要设置 Gemini API 密钥。
- 点击右上角的 **设置 (Settings ⚙️)** 图标。
- 输入您的 **Gemini API Key**。
- 若您还没有密钥，可以从 [Google AI Studio](https://aistudio.google.com/app/apikey) 免费申请。
- 若房间创建者已分享其密钥，您可能不需要输入自己的密钥。

> [!NOTE]
> 使用您自己的 API 密钥可确保拥有独立的额度与更佳的性能。

## 2. 会议室功能 👥
- **建立房间**：点击“建立新房间”即可开始对话。
- **加入房间**：输入房间 ID 或扫描他人分享的 QR Code。
- **分享**：点击 **分享 (Share)** 按钮可复制邀请链接或显示 QR Code。

## 3. 翻译与语音 🎙️
- **启动识别**：点击麦克风图标即可开始实时语音转文字与翻译。
- **双语显示**：系统会同时显示原文与翻译后的文字。
- **语音输出**：开启“语音输出”功能，即可听到高质量 AI 语音朗读翻译内容。

## 4. 进阶功能 🛠️
- **管理员设置**：自定义标题、调整 AI 反应速度及选择偏好的语音类型。
- **清除记录**：房间拥有者可以一键清除所有对话记录以重新开始。
- **深色模式**：可在浅色与深色模式间切换，提供舒适的阅读体验。
