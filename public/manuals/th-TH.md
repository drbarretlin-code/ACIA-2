# ACIA-2 ระบบล่าม AI อัจฉริยะแบบเรียลไทม์ - คู่มือการใช้งาน

ยินดีต้อนรับสู่ ACIA-2 แพลตฟอร์มการแปลและทำงานร่วมกันในการประชุมแบบเรียลไทม์ระดับพรีเมียม

## 1. เริ่มต้นใช้งาน - Gemini API Key 🔑
เพื่อเปิดใช้งานการแปลด้วย AI อย่างเต็มรูปแบบ คุณต้องมี Gemini API Key
- ไปที่ **การตั้งค่า (Settings ⚙️)** ที่มุมขวาบน
- ป้อน **Gemini API Key** ของคุณ
- หากคุณยังไม่มี สามารถรับได้ฟรีจาก [Google AI Studio](https://aistudio.google.com/app/apikey)
- หากผู้สร้างห้องแชร์คีย์ของตนเอง คุณอาจไม่จำเป็นต้องป้อนคีย์ของคุณเอง

> [!NOTE]
> การใช้ API Key ของคุณเองจะช่วยให้คุณมีโควต้าเฉพาะและประสิทธิภาพที่ดีขึ้น

## 2. ห้องประชุม 👥
- **สร้างห้อง**: คลิก "สร้างห้องใหม่" เพื่อเริ่มเซสชัน
- **เข้าร่วมห้อง**: ป้อน Room ID หรือสแกน QR Code ที่ผู้อื่นแชร์
- **การแชร์**: คลิกปุ่ม **แชร์ (Share)** เพื่อคัดลอก URL คำเชิญหรือแสดง QR Code

## 3. การแปลและเสียง 🎙️
- **เริ่มการแปล**: คลิกไอคอนไมโครโฟนเพื่อเริ่มการแปลงคำพูดเป็นข้อความและการแปลแบบเรียลไทม์
- **การแสดงผลสองภาษา**: ระบบจะแสดงทั้งข้อความต้นฉบับและข้อความที่แปลพร้อมกัน
- **เอาต์พุตเสียง**: เปิดใช้งาน "เอาต์พุตเสียง" เพื่อฟังคำแปลที่อ่านออกเสียงโดย AI คุณภาพสูง

## 4. คุณสมบัติขั้นสูง 🛠️
- **การตั้งค่าผู้ดูแลระบบ**: ปรับแต่งชื่อส่วนหัว ปรับการตอบสนองของ AI และเลือกประเภทเสียงที่ต้องการ
- **ล้างบันทึก**: เจ้าของห้องสามารถล้างบันทึกการถอดเสียงทั้งหมดเพื่อเริ่มต้นใหม่
- **โหมดมืด**: สลับระหว่างโหมดสว่างและโหมดมืดเพื่อการรับชมที่สบายตา
