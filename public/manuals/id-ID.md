# TUC AI Real-Time Intelligent Interpreter - Panduan Pengguna

Selamat datang di TUC, platform terjemahan waktu nyata dan kolaborasi rapat premium Anda.

## 🌟 Skenario Aplikasi
- **Mode Tatap Muka**: Satu orang membuka aplikasi, mengatur dua bahasa, dan mengaktifkan terjemahan ucapan dan teks waktu nyata untuk percakapan.
- **Mode Rapat Jarak Jauh**: Lihat "Fungsi Ruang Rapat" di bawah untuk kolaborasi multi-pihak jarak jauh.
- **Mode Rapat Lokal**: Beberapa orang terhubung ke ruang rapat yang sama. Pengguna dapat mengaktifkan pemutaran audio atau sekadar menelusuri jendela dialog untuk komunikasi multibahasa.

## 1. 🔑 Memulai - Kunci API Gemini
Untuk mengaktifkan kekuatan penuh terjemahan AI, Anda memerlukan Kunci API Gemini.
- Buka **Pengaturan (Settings ⚙️)** di sudut kanan atas.
- Masukkan **Gemini API Key** Anda.
- Jika Anda belum memilikinya, Anda bisa mendapatkannya secara gratis dari [Google AI Studio](https://aistudio.google.com/app/apikey).
- Jika pembuat ruangan membagikan kuncinya, Anda mungkin tidak perlu memasukkan kunci Anda sendiri.

> [!NOTE]
> Menggunakan kunci API Anda sendiri memastikan Anda memiliki kuota khusus dan kinerja yang lebih baik.

## 2. 👥 Ruang Pertemuan
- **Buat Ruangan**: Klik "Buat Ruangan Baru" untuk memulai sesi.
- **Bergabung ke Ruangan**: Masukkan ID Ruangan atau pindai Kode QR yang dibagikan oleh orang lain.
- **Berbagi**: Klik tombol **Bagikan (Share)** untuk menyalin URL undangan atau menampilkan Kode QR.

## 3. 🎙️ Terjemahan & Suara
- **Mulai Menerjemahkan**: Klik ikon mikrofon untuk memulai ucapan-ke-teks dan terjemahan waktu nyata.
- **Tampilan Bilingual**: Sistem menampilkan teks asli dan teks terjemahan secara bersamaan.
- **Output Audio**: Aktifkan "Output Audio" untuk mendengar terjemahan dibacakan oleh suara AI berkualitas tinggi.

## 4. 🛠️ Fitur Lanjutan
- **Pengaturan Admin**: Sesuaikan judul header, sesuaikan responsivitas AI, dan pilih jenis suara yang disukai.
- **Hapus Transkrip**: Pemilik ruangan dapat menghapus semua transkrip untuk memulai dari awal.
- **Mode Gelap**: Beralih antara mode terang dan gelap untuk tampilan yang nyaman.
