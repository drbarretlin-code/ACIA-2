# ACIA-2 AI 實時智能翻譯系統 - 操作說明書

歡迎使用 ACIA-2，您的頂級實時翻譯與會議協作平台。

## 1. 開始使用 - Gemini API Key 🔑
為了啟動完整的 AI 翻譯功能，您需要設定 Gemini API 金鑰。
- 點擊右上角的 **設定 (Settings ⚙️)** 圖示。
- 輸入您的 **Gemini API Key**。
- 若您還沒有金鑰，可以從 [Google AI Studio](https://aistudio.google.com/app/apikey) 免費申請。
- 若房間建立者已分享其金鑰，您可能不需要輸入自己的金鑰。

> [!NOTE]
> 使用您自己的 API 金鑰可確保擁有獨立的額度與更佳的效能。

## 2. 會議室功能 👥
- **建立房間**：點擊「建立新房間」即可開始對話。
- **加入房間**：輸入房間 ID 或掃描他人分享的 QR Code。
- **分享**：點擊 **分享 (Share)** 按鈕可複製邀請連結或顯示 QR Code。

## 3. 翻譯與語音 🎙️
- **啟動辨識**：點擊麥克風圖示即可開始實時語音轉文字與翻譯。
- **雙語顯示**：系統會同時顯示原文與翻譯後的文字。
- **語音輸出**：開啟「語音輸出」功能，即可聽到高品質 AI 語音朗讀翻譯內容。

## 4. 進階功能 🛠️
- **管理員設定**：自定義標題、調整 AI 反應速度及選擇偏好的語音類型。
- **清除紀錄**：房間擁有者可以一鍵清除所有對話紀錄以重新開始。
- **深色模式**：可在淺色與深色模式間切換，提供舒適的閱讀體驗。
